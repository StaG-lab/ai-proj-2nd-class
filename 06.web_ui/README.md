* 루트 디렉토리에 `db_config.json` 추가 필요

* streamlit 가상환경 생성
    ```cmd
    conda create -n streamlit2 python=3.11
    conda activate streamlit2
    ```

* requirements 설치
    ```cmd
    pip install -r 06.web_ui/requirements.txt
    ```

* 앱 구동
    ```cmd
    streamlit run --client.showSidebarNavigation=False 06.web_ui/Home.py
    ```
